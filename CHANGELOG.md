# Changelog

All notable changes to the **Privacy Guardian** project will be documented in this file.

## [1.0.0] - 2025-12-24
### Added
- Initial release of **Privacy Guardian: Offline Edition**.
- **App Auditor**: Granular permission analysis and risk scoring.
- **Control Center**: Privacy profiles (Sleep, Work, Custom).
- **Dark Mode**: Premium glassmorphism UI.
- **Monetization**: Non-personalized AdMob integration and Ad-Free IAP.
- **Export**: CSV data export for privacy audits.
- **Documentation**: New public documentation website and portal.
